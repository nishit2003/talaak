# Talaak.com

A sarcastic divorce website with Hindi-English humor.

## Deploy to GitHub Pages

1. Create a new repository on GitHub (name it `talaak` or `talaak.com`)
2. Don't initialize with README, .gitignore, or license
3. Run these commands:

```bash
git remote add origin https://github.com/YOUR_USERNAME/talaak.git
git push -u origin main
```

4. Go to your repository on GitHub
5. Click **Settings** â†’ **Pages**
6. Under **Source**, select **Deploy from a branch**
7. Select **main** branch and **/ (root)** folder
8. Click **Save**
9. Your site will be live at: `https://YOUR_USERNAME.github.io/talaak/`

## Local Development

Just open `index.html` in your browser!
